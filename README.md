# Telegram Challenge Bot

Бот для семейного трекера привычек (челленджа).

## Возможности

- ✅ Отметка привычек через удобную клавиатуру
- ✅ Статистика выполнения привычек в виде календаря
- ✅ Хранение данных в SQLite базе данных
- ✅ Поддержка нескольких пользователей в одной группе
- ✅ Индивидуальные наборы привычек для каждого пользователя

## Структура проекта

- `main.py` - основной файл бота
- `database.py` - работа с SQLite базой данных
- `data.py` - функции для работы с данными (обертки над БД)
- `init_chat_data.py` - скрипт для инициализации данных конкретного чата
- `requirements.txt` - зависимости проекта

## Запуск

1. Создай виртуальное окружение:
```bash
python3 -m venv venv
source venv/bin/activate
```

2. Установи зависимости:
```bash
pip install -r requirements.txt
```

3. Создай `.env` файл в родительской директории с токеном:
```
BOT_CHALLENGE_TOKEN=твой_токен_от_BotFather
```

4. Запусти бота:
```bash
python main.py
```

## Технологии

- Python 3.10+
- aiogram 3.13.1
- python-dotenv 1.0.1
- aiosqlite 0.19.0

## База данных

Бот использует SQLite для хранения данных:
- `habits_tracker.db` - файл базы данных (создается автоматически)
- Таблицы: `tracker_entries`, `habits`, `users`

## Команды бота

- `/start` - запуск бота и показ клавиатуры
- `/init_data` - инициализация тестовых данных для группы
- `/chat_id` - показать ID чата/группы
- `/my_id` - показать ID пользователя
- `/get_members` - список участников группы (только для групп)
